# ✅ QA Practices – E2E Ecommerce Website

## Coding Standards
- Follow PEP8 for Python syntax
- Use meaningful test names and comments
- Maintain modular design (POM principle)

## Review Process
- Peer review before merge
- PR approval required for `main` branch

## Best Practices
- Keep tests independent
- Avoid hard-coded data
- Use assertions for validations
- Regular cleanup of reports and logs
